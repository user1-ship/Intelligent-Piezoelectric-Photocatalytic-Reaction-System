系统概述

智能压电光催化控制平台是一个集实时监控、数据分析、智能控制和故障诊断于一体的工业控制系统。系统主要包含以下核心模块：

  • 数据中心 实时
  • 历史数据 分析
  • 系统配置 设置
  • 数字孪生 仿真
  • 强化学习 智能
  • 故障诊断 维护
常见问题

系统默认管理员账号：admin，密码：admin123。首次登录后请及时修改密码。

普通操作员账号需要由管理员在系统配置 → 安全设置 → 用户管理中创建。

数据中心的数据更新频率取决于传感器的采样频率设置：

  • 实时数据：默认5秒更新一次
  • 传感器原始数据：根据配置的采样频率（1Hz-50Hz）
  • 图表刷新：10秒更新一次

您可以在系统配置 → 传感器配置中调整各传感器的采样频率。

历史数据导出步骤如下：

  1. 进入历史数据页面
  2. 设置查询条件（时间范围、数据类型、数据频率）
  3. 点击查询数据按钮
  4. 点击导出数据按钮
  5. 选择导出格式（CSV或Excel）

导出的数据文件包含所有查询结果，文件命名格式为：数据导出_YYYYMMDD_HHMMSS.csv

是的，系统配置功能需要管理员权限。用户分为三个级别：

角色 权限
管理员 所有权限，包括系统配置、用户管理
操作员 数据中心查看、历史数据查询、部分配置调整
查看者 仅查看权限，不能修改任何配置

操作日志查看方法：

  • 在系统配置 → 安全设置中启用操作审计日志功能
  • 系统会自动记录所有重要操作，包括：登录/注销、配置修改、数据导出等
  • 日志文件保存在/logs/operation.log
  • 管理员可以在系统配置 → 备份与恢复中下载日志文件

传感器校准步骤：

  1. 进入系统配置 → 传感器配置
  2. 在传感器列表中找到需要校准的传感器
  3. 点击对应的校准按钮
  4. 按照校准向导进行操作（通常需要输入标准值）
  5. 保存校准参数

注意：部分传感器校准需要专业设备，建议由专业人员操作。
